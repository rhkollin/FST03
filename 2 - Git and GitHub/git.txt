- Git version

git --version


- Configure Git
git config --global user.name "rhkollin"
git config --global user.email "rholghadokollin@gmail.com"
git config user.name

- Legends
U - Untracked
A - Added
M - Modified

- Initialize repository
git init

- Git Three Stage

- Check the status
git status

- Add file to the staging area
git add .

- Create a commit
git commit -m "This is a description"

- Show the history
git log

- Shorter history
git log --oneline
